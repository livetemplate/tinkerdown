# Feature Requests

```html
<form lvt-persist="requests">
  <input name="title" placeholder="Feature title">
  <textarea name="description"></textarea>
  <select name="priority">
    <option>Nice to have</option>
    <option>Important</option>
    <option>Critical</option>
  </select>
  <button>Submit</button>
</form>
```

```lvt
| Feature | Priority |
|---------|----------|
{{range .requests}}
| {{.title}} | {{.priority}} |
{{end}}
```

Use any language

Build a UI for your scripts in 30 seconds

Don't know Go? No problem. LivePage's polyglot mode lets you use Python, Bash, Node, or any language as a data source. Write a script that outputs JSON, and LivePage handles the rest.

Python Bash Node.js Ruby Rust Any CLI

    type: exec
    cmd: python scripts/fetch_sales.py
    interval: 60s
